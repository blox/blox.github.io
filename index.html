What is Amazon ECS?

Amazon EC2 Container Service (Amazon ECS) is a container management service for deploying and scaling Docker containers in production. Amazon ECS makes it easy to securely run containerized applications at high levels of scale and fault tolerance. You just add the server capacity you need to your cluster and upload your container images. Amazon ECS takes care of the rest.

It runs your containers across a cluster of servers and takes care of load balancing and scaling your containers. There is no container management software to install and operate because it is all provided to you as a service, making it great for production applications.

The scheduling frameworks that make up Blox are also available to Amazon ECS users without having to build, deploy and manage any custom scheduling logic.   Learn more >
